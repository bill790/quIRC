The setting's name in the rc file (typically ~/.quirc/rc), if settable there.
cmdline
The setting's name on the command line, if settable there.
/set
The name to use with the /set command. If the setting is an INT, use /set <name> <value>; omitting <value> sets to the default value. If the setting is a BOOL, use /set <name> to enable, and /set no-<name> to disable (you can also use other forms, such as supplying a <value> of + or 1 to enable and - or 0 to disable).
description
A short description of the setting, as used in messages produced by /set.
